# DatasetLLM
An AI-powered application for extracting data from text

This project aimed to harness the power of LLMs to extract useful data from the vast unstructured textual data found on the internet. I used the Llama2 7B model published by Meta 
and prompted it to extract data based on user preferences, such as sentiment, summary, and more. The model currently returns the extracted data as a Pandas Dataframe, and provides a
reasonably accurate dataset given the relative simplicity of the program.
